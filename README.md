# 씨앗 9기 DB 미니 프로젝트 

## 📅 프로젝트 기간
2025/04/01 ~ 2025/04/04

## 팀원 및 역할
- **정지훈**: 팀장, 퀴즈 API, 프로젝트 설계, ERD 및 DB관리
- **전병윤**: 게임 화면(View), 와이어프레임 디자인
- **최은정**: 퀴즈 데이터, 요구사항 명세서, 활동 API
- **박신형**: 아이템 데이터, 상점 아이템목록 조회, 상점 및 아이템 API
